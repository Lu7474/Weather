# Это приложение для отображения погоды в выбранном городе с использованием Flask и API OpenWeatherMap.
## Пользователи могут ввести название города, и приложение отобразит текущую погоду, включая температуру и описание погоды.

### 1. Получите API-ключ на сайте OpenWeatherMap(https://openweathermap.org/)
### 2. Зарегистрируйтесь на OpenWeatherMap.
### 3. Перейдите в раздел «API» и получите свой личный API-ключ.(если не работает мой)

### 4. Запуск приложения
```bash
python run.py
```
Приложение будет доступно по адресу: http://127.0.0.1:5000/
